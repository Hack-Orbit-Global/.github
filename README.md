# 🚀 Hack Orbit

### Build – Contribute – Orbit Together

Hack Orbit is an open-source builder community where developers collaborate on real projects, participate in hackathons, and grow by building in public.

---

## 🌍 About Hack Orbit

Hack Orbit is designed for developers who want to:

- Build real-world projects  
- Contribute to open source  
- Learn by doing, not just watching  
- Collaborate with other developers  

We focus on creating a space where **builders grow together**.

---

## 🧱 What We Do

- 🚀 Build open-source projects  
- 🏆 Organize hackathons and challenges  
- 🤝 Help developers collaborate  
- 📚 Promote learning through real work  

---

## 🚀 Projects

Projects under Hack Orbit:

- **TaxGuide** – Simplifying taxes with easy tools and guides
